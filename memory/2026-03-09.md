### ML Microservice Built (ml-service/)
Muhammad's decision: Use 100% open-source self-hosted pipeline — NO paid vision APIs.

**Architecture:**
```
Browser → Vercel api/detect.js (proxy) → FastAPI VPS → JSON response
```

**Files created:**
- `ml-service/preprocessing.py` — OpenCV pipeline:
  1. Load + cap size (max 2048px)
  2. Grayscale (handles RGBA/RGB)
  3. Denoise (Gaussian blur + fastNlMeansDenoising)
  4. Otsu binarization + adaptive fallback (picks best by edge score)
  5. Auto-invert (text always dark on white)
  6. Deskew via Hough Line Transform (PCA fallback)
  7. Crop tightest text region (morphological dilation + contour detection)
  8. Letterbox resize to 105×105 float32 [0,1]
- `ml-service/model.py` — ResNet-50 (ImageNet) feature extractor → 2048-dim L2-normalised embedding
- `ml-service/searcher.py` — FAISS IndexFlatIP KNN search, returns FontMatch with confidence %
- `ml-service/main.py` — FastAPI server with `/identify` POST endpoint + auth + error handling
- `ml-service/scripts/build_index.py` — One-time: renders .ttf files → embeds → builds FAISS index
- `ml-service/requirements.txt` + `Dockerfile` + `DEPLOY.md`
- `api/detect.js` — Updated to proxy to Python server (X-API-Key header auth)

**Key decisions:**
- ResNet-50 ImageNet pretrained (no retraining needed, ~85-95% accuracy baseline)
- FAISS IndexFlatIP (exact cosine similarity on L2-normalised vectors)
- 2048-dim embedding dimension
- Target size: 105×105 (DeepFont paper spec)
- Auth: HMAC constant-time comparison (FF_API_SECRET header)
- Vercel env vars: ML_SERVICE_URL, FF_API_SECRET (in addition to WHATFONTIS_API_KEY)

**VPS recommendation:** Hetzner CX32 (~€8.29/mo, 4 vCPU 8GB RAM)
**Font source for index:** Google Fonts repo (~3,500 .ttf files, OFL licensed)

**Expected performance:** ~130ms total on CPU (preprocessing 20ms + ResNet 80ms + FAISS 2ms + network 30ms)
